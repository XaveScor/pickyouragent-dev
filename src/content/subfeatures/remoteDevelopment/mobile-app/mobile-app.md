Code from anywhere with mobile access.

Access and monitor your coding sessions from mobile devices. Start tasks on the go, monitor progress, and receive notifications when work completes. Perfect for checking PRs, reviewing code changes, or kicking off background tasks while away from your desk.

**Supported agents:**

- Claude Code (iOS app available - https://code.claude.com/docs/en/claude-code-on-the-web)
- Cursor (mobile agent available - https://cursor.com)
- Codex (ChatGPT mobile app integration - https://chatgpt.com/pricing/)

**Partially supported agents:**

- OpenCode (client/server architecture supports mobile clients, but no native mobile app available - https://opencode.ai/docs/server/)

**Not verified yet:**

- Kilo Code
- Junie
