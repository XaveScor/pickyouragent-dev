Run the agent on your own infrastructure.

Deploy the agent on servers you control for security, compliance, or privacy requirements. This includes self-hosted deployments, enterprise infrastructure integration, and custom cloud provider setups. Open source solutions allow you to host the entire agent stack on your own servers, while enterprise solutions offer deployment through major cloud providers.

**Supported agents:**

- Kilo Code (open source, can self-host)
- OpenCode (https://opencode.ai/docs/server/)

**Partially supported agents:**

- Claude Code (cloud provider deployments available - Amazon Bedrock, Google Vertex AI, Microsoft Foundry, Anthropic Console - https://code.claude.com/docs/en/third-party-integrations)

**Not supported agents:**

- Cursor (fully managed service only)
- Codex (fully managed service by OpenAI)

**Not verified yet:**

- Junie
