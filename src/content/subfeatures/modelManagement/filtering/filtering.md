Hide or show models from the list.

The agent allows you to hide/show the models from the model list.

**Supported agents:**

- Cursor
- OpenCode (type ctrl+x+m and select your favourite model)

**Not supported agents:**

- Claude Code
- Codex
- Kilo Code
- Junie (Uses automatic model selection)

**Not verified yet:**

- None
